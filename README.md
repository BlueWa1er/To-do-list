# To-Do List (Android)

---

To-Do List, створений на Kotlin з використанням Jetpack Compose.

Додаток дозволяє працювати зі списком завдань

---

### Основний функціонал 
- Додавання нових завдань
- Відображення списку завдань
- Вгорі екрана - поле для введення тексту і кнопка "Додати"
- Коли користувач вводить текст у поле і натискає кнопку, нова справа додається до списку і відображається на екрані

---

### Додатковий функціонал (за власним розсудом)
- Анімації при додаванні та видаленні завдань
- Плавні UI-анімації для всього списку
- Невиконані завдання автоматично відображаються зверху
- Видалення завдань
- Позначення завдань як виконаних
- Покращений зовнішній вигляд інтерфейсу
---

### Технології
- Kotlin
- Jetpack Compose
- Android Studio

---
### Примітки
- Дані не зберігаються після закриття додатку

# To-Do List (Android)

---

To-Do List, created in Kotlin using Jetpack Compose.

The app allows you to manage a list of tasks.

---

### Main Features
- Add new tasks
- Display the list of tasks
- At the top of the screen: a text input field and an "Add" button
- When the user enters text in the field and clicks the button, the new task is added to the list and displayed on the screen

---

### Additional Features (optional)
- Animations when adding and deleting tasks
- Smooth UI animations for the entire list
- Incomplete tasks are automatically displayed at the top
- Task deletion
- Mark tasks as completed
- Improved UI design

---

### Technologies
- Kotlin
- Jetpack Compose
- Android Studio

---

### Notes
- Data is not saved after closing the app
